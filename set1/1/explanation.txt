COMMANDS

$ g++ -Wall -std=c++23 -Werror -o hello hello.cc	//to link
$ g++ -Wall -std=c++23 -Werror -c hello.cc		//to compile

DESCRIPTIONS

the object file is file type used to link the programme to another c++
programme. the executable file is the file that one can execute in their
operating system to see the output of the programme.

Hello World
